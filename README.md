# A Day in the Life of a Biomedical Scientist

## Overview

This project is an interactive web experience that allows users to explore the role of a **Biomedical Scientist working in the NHS**.

The website simulates part of a biomedical scientist’s shift, where the user must analyse a blood sample and help prepare a patient for a life-saving transfusion. After completing the activity, the user learns more about the career, including responsibilities, key skills, and the qualifications pathway.

This project was designed to raise awareness of biomedical science careers within the NHS and demonstrate how important laboratory scientists are in patient care.

---

## Features

* Interactive **multi-screen experience**
* Simulated **NHS laboratory task**
* Blood sample **analysis scenario**
* Information about the **Biomedical Scientist career**
* Qualification pathway explanation
* Restartable experience so users can try again

---

## How to Use

1. Open the website using the GitHub Pages link.
2. Click **Start Your NHS Shift**.
3. Watch the shift clock reach **18:00:00**.
4. Analyse the blood sample and choose the correct blood type.
5. Continue through the screens to learn about the biomedical scientist career.
6. Click **Restart** if you want to experience it again.
